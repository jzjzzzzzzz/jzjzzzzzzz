## What I build

I like projects that are more than demos: they start from a real problem, turn it into a structured model, validate the output, and then become something people can actually use.

My current interests include:

- AI agents that can reason through domain-specific tasks instead of only chatting
- CAD intent parsing: natural language → structured design intent → validation → model generation
- Safe desktop automation with OCR, status checks, dry runs, and audit logs
- Research software for biology, environmental data, and simulation-heavy projects
- Computer vision tools for screen recognition and decision support
- Music technology tools for sheet-music conversion and transposition

## Current focus

```text
Natural language prompt
        ↓
Structured intent schema
        ↓
Validation rules
        ↓
Generated artifact
        ↓
Tests, benchmarks, and real use
```

Right now, I am especially interested in building domain-specific agents: tools that understand one field deeply enough to reject bad prompts, ask for missing constraints, produce structured outputs, and verify their own results.
